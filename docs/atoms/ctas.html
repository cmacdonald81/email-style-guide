Button

Atmosphere

Vocabulary

Button

Syntax

The solid green button is the default, and it can be centered or left aligned.

Note: If adding to a multi-column layout, you'll need to add colspan="" (add the number of columns you want the button to span in those quotes)

Semantics

Use the button for primary actions. It makes the Call to Actions (CTA) stand out more than a text link.

  • CSS

    </td>
  </tr>
</table>
