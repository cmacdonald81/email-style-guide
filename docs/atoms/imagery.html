Words words words about when to use a photo vs. an image vs. a GIF vs. a video

Best Practices

  • Keep the aspect ratio. Don't warp the images. Constrain proportions to retain the aspect ratio of the original image.
  • Make it retina-ready. Save images at 2x. This ensures retina-ready images. The images will be scaled down to fit.
  • Don't use background images. They don't work in Microsoft Outlook or in Gmail.
  • Plan for text-only emails. Prepare for the emails to be viewed without images. Include alt text that describes the image. Clients such as Outlook, AOL, and Yahoo have images blocked by default, and in other clients users can turn this on in their preferences.
  • Images can be links
  • Keep images under x. Large file sizes take too long to load.
  • Don't put text in images. Text is put outside images in order to gain control over readability at small screen sizes.

Full Width

Vocabulary

Full Width Image

Syntax

Full width photographs span the width of the white section.

Semantics

These are used when you want a section of content to stand out as a featured section.

  • HTML
  • CSS
<td class="text-center-xs image-full" align="center">
          <img width="600" src="image-full.png" alt="Placeholder">
      </td>
.image-full img {
        width: 600px;
        min-width: 100%;
        max-width: 100%;
        height: auto;
        border: 0;
        line-height: 100%;
        outline: none;
        text-decoration: none;
        text-align: center;
  }

Inline

Vocabulary

Inline Image

Syntax

Inline images sit within the width of the white section, but show padding on the left and right side.

Semantics

These are used to reinforce or expand upon the supporting text.

  • HTML
  • CSS
<img src="image-full.png" alt="Placeholder"></img>
img {
        width: 100%;
        max-width: 512px;
      }

Attribution

Vocabulary

Attribution, Image Source, Image Credit

Syntax

Attribution lines sit below the right bottom corner of the image.

Semantics

Some images we use require attribution. The attribution line conveys the image source without competing with the other content in the email.

Guidelines
  • Don't embed videos directly into emails. Embedded videos aren't widely supported across email clients. Instead, include a video still and then link that picture to the video so that the video plays in a new tab.
  • Use linked inline image. Add the video still as a regular inline image and wrap it in a link to the video.
  • Make it look like an app video. Video previews should look like the videos in the Treehouse library. Don't use video previews automatically generated by MailChimp or other ESPs.
  • Choose eye-catching video stills. Choose a video still that doesn't have any text in it and ideally features a person. People are more attracted to images with people.
  • Use this handy tool. You can use youtubescreenshot.com to easily generate a video still. Choose a moment in the video that features people (ideally) or a catchy visual that represents the gist of the video. Don't choose a frame that includes text.
